## vue-meituan-learning

> 慕课网项目，主要技术栈：Vue.js 2.5 + Koa2 + Nuxt.js + Element + MongoDB + Redis，SSR全栈项目。

### 项目构建

vue init nuxt-community/koa-template <project-name>
cd <project-name>
npm install  // 安装依赖
npm run dev  //启动项目

### MongoDB启动

以管理员模式打开命令管理器，执行net start MongoDB
数据可视化工具 RoBo 3T


### Redis启动

Redis.server.exe
